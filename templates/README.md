# SDLC AI Templates

This directory contains blank templates for all 8 phases of the SDLC AI governance process.

## Usage

1. **Copy** the relevant template pairs (md/json) to your `architecture/` directory.
2. **Rename** them to the standard format (e.g., `00-triage.md`).
3. **Fill in** the details as per the workflow protocol.
4. **Validate** the JSON against the schema.

## Quick Reference
- **Phase 0 (Triage)**: `00-triage.template.{md,json}`
- **Phase 1 (Context)**: `01-context.template.{md,json}`
- **Phase 2 (Design)**: `02-design.template.{md,json}`
- **Phase 3 (Security)**: `03-security.template.{md,json}`
- **Phase 4 (Decision)**: `04-decision.template.{md,json}`
- **Phase 5 (Plan)**: `05-plan.template.{md,json}`
- **Phase 6 (Execute)**: `06-report.template.md` (Markdown only)
- **Phase 7 (Verify)**: `07-verify.template.md` (Markdown only)
